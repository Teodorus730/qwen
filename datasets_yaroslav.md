## Датасеты для обучения и оценки LLM
Англоязычные датасеты для Pretrain и SFT
### [Fineweb](https://huggingface.co/datasets/HuggingFaceFW/fineweb) / [Fineweb-edu](https://huggingface.co/datasets/HuggingFaceFW/fineweb-edu)

Описание: Масштабный веб-корпус, извлеченный из дампов Common Crawl. Прошел жесткую дедупликацию и эвристическую фильтрацию. Подмножество FineWeb-Edu дополнительно отфильтровано с помощью классификаторов для сохранения текстов с наивысшей образовательной ценностью.  

Язык: Английский.

Особенности: Базовый датасет является одним из стандартов для предварительного обучения современных открытых моделей.  

### [dclm-baseline-1.0](https://huggingface.co/datasets/mlfoundations/dclm-baseline-1.0)

Описание: Высококачественный дистиллированный набор веб-данных, собранный в рамках проекта DataComp for Language Models. Тексты проходят через ML-классификаторы, что позволяет радикально снизить процент информационного мусора.  

Язык: Английский.

Особенности: Отличается высоким соотношением полезного сигнала к шуму.

### [the-stack-v2](https://huggingface.co/datasets/bigcode/the-stack-v2)

Описание: Крупнейший репозиторий программного кода, собранный с GitHub под открытыми лицензиями.  

Язык: Мультиязычный (языки программирования).

Особенности: Содержит около 5,45 миллиарда файлов. Критически важен для внедрения логического мышления и структурного вывода в веса LLM.  

И также small-версия [the-stack-v2-train-smol-ids](https://huggingface.co/datasets/bigcode/the-stack-v2-train-smol-ids)


### NuminaMath-CoT

Ну и этот датасетик тоже добавил бы для обучения, тк тут пошаговое решение сложных мат.решений (COT). И он как раз будет важен для раннего внедрения рассуждений

### [Cosmopedia](https://huggingface.co/datasets/HuggingFaceTB/cosmopedia)
Описание: Крупная коллекция синтетических учебников, блогов и историй, сгенерированных высокопроизводительными моделями.  

Язык: Английский.

Особенности: Содержит более 31 миллиона документов, которые гарантируют безупречно грамматически выверенный и фактологически плотный обучающий сигнал.  

### [FineMath](https://huggingface.co/datasets/HuggingFaceTB/finemath)
Описание: Тщательно курированные страницы с математическим уклоном, извлеченные из Common Crawl.  

Язык: Английский.

Особенности: Содержит около 48 миллионов документов, предназначенных для улучшения навыков математического рассуждения моделей.  

### [OpenThoughts-3](https://huggingface.co/datasets/open-thoughts/OpenThoughts-3)
Набор высококачественных запросов и многошаговых цепочек рассуждений (Chain-of-Thought), покрывающий разнообразные математические и логические задачи. В среднем содержит более 20 шагов рассуждения на каждый пример.

Язык: Английский

## Русскоязычные бенчмарки и NLU датасеты
### MERA (Multimodal Evaluation for Russian-language Architectures)
Описание: Открытый бенчмарк для комплексной оценки фундаментальных моделей на русском языке. Включает в себя 21 задачу в инструктивном формате.  

Язык: Русский.

Особенности: Покрывает навыки логики, здравого смысла, алгоритмического вывода и знаний о мире. Включает уникальные сеты, такие как вопросы из ЕГЭ, ruModAr (математика), а также диагностические наборы по этике и токсичности (ruDetox, ruHateSpeech).  

### SAGE Datasets (RUSpellRU, MultidomainGold)
Описание: Набор открытых параллельных датасетов, специально созданных для задачи коррекции орфографии и пунктуации.  

Язык: Русский.

Особенности: Содержит более 13 тысяч примеров с ручной разметкой, собранных из более чем 10 текстовых доменов (включая веб, новости, литературу и Github). Позволяет обучать модели исправлять ошибки, сохраняя при этом изначальную эмоциональную палитру текста.
